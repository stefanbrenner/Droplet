# Droplet ![Release](https://img.shields.io/badge/Version-v2.0.0-green.svg) [![Donate](https://img.shields.io/badge/Donate-PayPal-green.svg)](https://www.paypal.com/donate/?hosted_button_id=BV5H54EQ7T6ZL)

Droplet is an Open Source Project for the microcontroller-based High-Speed-Photography with special attention to Waterdrop Photography and Liquid Art.
More information can be found on www.droplet.at

![Droplet Screenshot](https://www.stefanbrenner.at/images/b/0/5/2/1/b0521213436a1e26e14d0c5e2aea7c485312e7d7-droplet-screenshot2x.png)

## Features
* Configuration of an unlimited (*) amount of devices (valves, flashes, cameras)
* Configuration of an unlimited (*) amount of actions per device
* Input of times in milliseconds
* Saving and loading of configurations
* Automatic creation of XMP Metadata Files
* English and German UI
* Platform independent: available for Windows, Mac OS X and Linux
* Open Source: the Source Code is free available and is licensed under the GNU General Public License v3.0
* Easy changing and expanding of the used microcontroller.  
  (Version 0.1 supports just [Arduino](https://arduino.cc/?target=_blank&rel=noreferrer))
* Easy changing of the communication protocol that is used to communicate with the microcontroller.  
  (Version 0.1 contains a Droplet Arduino-Sketch. Additionally, Nicolai Korffâ€™s [crazyMachine](http://crazymachine.nicolai-korff.de/?target=_blank&rel=noreferrer) is supported)

(*) technical limit defined by the used microcontroller
